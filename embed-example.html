🔗 Embedding ConceptARC Visualizer

Example of how to embed the visualizer in your own website

Simple Embedding

To embed the ConceptARC Visualizer in your own website, simply include the CSS and JavaScript files, then create a container element:

<!-- Include the visualizer files -->
<link rel="stylesheet" href="visualizer.css">
<script src="visualizer.js"></script>

<!-- Create a container for the visualizer -->
<div id="my-visualizer"></div>

<!-- Initialize the visualizer -->
<script>
    const visualizer = new ConceptARCVisualizer('my-visualizer', {
        cellSize: 30,
        maxCellSize: 60,
        minCellSize: 8
    });
</script>

Configuration Options

The visualizer accepts several configuration options:

const visualizer = new ConceptARCVisualizer('container-id', {
    cellSize: 30,        // Default cell size in pixels
    maxCellSize: 60,      // Maximum cell size for scaling
    minCellSize: 8        // Minimum cell size for scaling
});

API Methods

The visualizer provides several methods for programmatic control:

// Load data programmatically
visualizer.loadCSV(file);

// Navigate programmatically
visualizer.navigateLeft();
visualizer.navigateRight();

// Refresh the display
visualizer.refresh();

// Update status
visualizer.updateStatus('Custom message', 'success');

Styling Customization

You can customize the appearance by overriding CSS classes:

/* Custom styling */
.visualizer-container {
    border: 2px solid #your-color;
    border-radius: 12px;
}

.panel h3 {
    color: #your-color;
    font-weight: bold;
}

.btn-primary {
    background: #your-color;
    border-color: #your-color;
}
